#  Spotify & YouTube Data Analysis (Python)

##  Project Overview
This project performs exploratory data analysis (EDA) on Spotify and YouTube data to uncover insights about music trends, artist popularity, and user engagement.

The analysis focuses on metrics such as views, likes, comments, and streaming performance across different artists and album types.

---

##  Objectives
- Analyze artist popularity based on total views
- Compare engagement metrics (likes, views, comments)
- Identify top-performing artists and channels
- Understand trends across different album types (album, single, compilation)

---

##  Tools & Technologies Used
- Python
- Pandas (Data Cleaning & Analysis)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook

---

##  Analysis Performed
- Data cleaning and preprocessing
- Grouping data by artist and calculating total views
- Sorting and identifying top artists
- Visualization using bar charts and pie charts
- Comparative analysis of album types and engagement metrics

---

##  Key Insights
- Top artists like Ed Sheeran and CoComelon generate the highest views
- Album-type content contributes significantly to total engagement
- Views dominate compared to likes and comments across all categories
- Certain channels consistently outperform others in terms of reach
